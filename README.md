## 📖 Course Overview

This course introduces students to the fundamentals of web technology — from how the internet works, to building static and dynamic web pages with HTML, CSS, JavaScript, and Bootstrap. By the end, students will be able to design, develop, and deploy responsive websites with form validation and basic database connectivity.

**Learning Path:**
- Internet fundamentals (TCP/IP, DNS, HTTP, HTTPS)
- HTML static web pages (links, images, forms, tables)
- HTML5 semantic elements and new APIs
- CSS styling (selectors, layouts, responsive design)
- JavaScript interactivity and form validation
- Bootstrap responsive framework
- Web forms, basic authentication, and localStorage

## 📋 Each Lecture File Contains

Every file in `2-Content/` units (1–7) follows the same structure:

- **Learning Objectives** — What you'll know/be able to do
- **Theory Sections** — Concepts with real-world analogies

## ✅ How to Navigate This Course

### If You're Following the Day-by-Day Schedule
1. Check `Lecture-Plan.md` for today's date and topic
2. Open the corresponding file in `2-Content/` (e.g., `2-Content/2-HTML-Fundamentals/06-...md`)
3. Read the theory, work through the code examples
4. Complete the practical lab exercise
5. Submit via GitHub Classroom (if it's a graded lab)

### If You Want to Jump to a Specific Topic
1. Browse the unit folders in `2-Content/` (numbered 1–7 by topic)
2. Each folder has files numbered sequentially (01, 02, 03...)
3. Open the file for the topic you want

### If You're Preparing for Exams
1. Refer to `1-Curriculum/25BCA060T_Web_Technology.md` for official exam topics
2. Read the **Summary Tables** in each lecture file
3. Answer the **Self-Assessment Questions** at the end of files
4. Review the self-study materials in `2-Content/8-Self-Study/`

### If You're Stuck on a Problem
1. Check `2-Content/0-Instructions-and-Prerequisites/07-Troubleshooting-and-Help.md` first
2. Common Git/GitHub issues are documented there
3. If it's about course content, re-read the theory section with analogies
